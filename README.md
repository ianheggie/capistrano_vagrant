# Capistrano_Vagrant

Automatically configure capistrano with vagrant machines

## Installation

Add this line to your application's Gemfile:

    gem 'capistrano_vagrant'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install capistrano_vagrant
